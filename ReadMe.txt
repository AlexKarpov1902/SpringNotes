Веб-приложение для ведения заметок.


Для развертывания приложения необходимо выполнить следующие операции:
 
   
1. Разархивируйте файл в отдельную папку.
   
2. Перейдите в папку, в которой находится файл pom.xml.
 
3. Для запуска приложения введите команду                               mvn spring-boot:run
  
4. Перейдите в браузере по адресу http://localhost:8080/





При посещении страницы браузера по вышеуказанному адресу будет произведен запрос авторизации пользователя.
Для входа в приложение необходимо ввести имя пользователя и пароль ("user1" и "password1"). Для храниния данных 
используется встроенная база данных H2. С помощью функционала размещенного на главной странице можно добавлять 
записи с заметками, редактировать, удалять и производить поиск записей по названию заметки или её содержимому.
Настройки приложения находятся в файле src\main\resources\application.properties.
